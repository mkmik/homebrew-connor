# Homebrew Tap for Connor

This is a [Homebrew](https://brew.sh) tap for installing [Connor](https://github.com/mkmik/connor), a developer tools macOS application.

## Installation

```bash
brew tap mkmik/connor
brew install --cask connor
```

Or install directly:

```bash
brew install --cask mkmik/connor/connor
```

## Requirements

- macOS 14.0 (Sonoma) or later

## Updating

```bash
brew upgrade --cask connor
```

## Uninstalling

```bash
brew uninstall --cask connor

# To also remove preferences and cache:
brew uninstall --cask --zap connor
```
